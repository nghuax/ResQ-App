# ResQ Mobile App

A standalone static mobile app prototype for ResQ based on the Figma `Mobile App` section.

## Run locally

Open [index.html](./index.html) with the Live Server extension.

## Included screens

- `Services`
- `Nearby Garage`
- `Order`
- `Profile`

## Included product states

- Service filters and request sheet
- Nearby garage selection and dispatch
- Idle, active, completed, and cancelled order flows
- Profile layers for personal info, vehicles, notifications, security, and logout

No build step is required. Everything runs with plain `HTML`, `CSS`, and `JavaScript`.
